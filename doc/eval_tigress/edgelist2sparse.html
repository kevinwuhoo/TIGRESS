DESCRIPTION 

 Written by G. Stolovitsky and R. Prill for DREAM challenge evalution. 
 A = edgelist2sparse(E) transforms two-column matrix representaing edges
 (E) into sparse adjacency matrix A. 
 
 See also : GRNInferenceEvaluation
